## 🧠 Sobre mim

Sou desenvolvedor com foco em backend e análise de dados, apaixonado por resolver problemas reais através da tecnologia.

Atualmente atuo na área de logística, desenvolvendo soluções voltadas para automação de processos, controle operacional e aumento de visibilidade nas operações.

Sou criador do **DNATransportes**, um sistema de gestão logística que integra pedidos, rotas, rastreamento e análise de dados em uma única plataforma.

---

## 🚚 Projeto em destaque

### 🔥 DNATransportes

Sistema desenvolvido para transformar processos logísticos manuais em uma operação inteligente, conectada e orientada por dados.

💡 Principais pontos:
- Gestão de pedidos e rotas  
- Rastreamento de entregas  
- Importação de NF-e (XML)  
- Dashboard com indicadores  
- Integração com API e mobile  

👉 [Acessar repositório](https://github.com/romullo-dev/DNATransportes)
